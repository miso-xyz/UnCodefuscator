# UnCodefuscator
Deobfuscator for Codefuscator (tested on v0.1.2)</br></br>
Note: <a href="https://github.com/0x421/Codefuscator">Codefuscator</a> has a lot of similarities with <a href="https://github.com/call-042PE/VaporObfuscator">VaporObfuscator</a>, this deobfuscator is based on <a href="https://github.com/miso-xyz/DeVapor">DeVapor</a>.</br>(woudn't be surprised if its basically a mod of <a href="https://github.com/call-042PE/VaporObfuscator">VaporObfuscator</a> lmao)

# Is able to
<pre>
- Remove AntiDe4dots
- Decode Strings
- Retrieve System Types Names (MySettings, MyProject etc...)
- Junk Methods & Classes

ToDo:
- Control Flow (cannot apply protection due to it requiring a stupid amount of RAM)
- Numbers Obfuscation (obfuscator crashes)
</pre>

# Usage
Drag & Drop to deobfuscate</br>
TIP: you can add `-showAll` to the argument list to see everything happening (make sure the file to deobfuscate is the first argument)
